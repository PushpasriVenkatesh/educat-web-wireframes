****EduCat – Web Platform Wireframes****

This repository contains low-fidelity wireframes for an online educational learning platform named EduCat.
The wireframes were created using Figma and represent the basic structure, layout, and navigation flow of the website.

****Project Overview****

EduCat is designed as a simple and user-friendly platform where students can browse courses, view details, and manage their learning journey.
These wireframes focus on clarity, usability, and minimalistic layout decisions before moving into high-fidelity UI design.

**Included Wireframes**

The following screens are included in this wireframe set:

> Home Page

> Course Listing Page

> Course Details Page

> User Login / Sign-Up Pages

> Dashboard View

> Settings / Profile Page
